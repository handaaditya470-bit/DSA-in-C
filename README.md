# DSA in C++

This repository contains implementations of fundamental **Data Structures and Algorithms (DSA)** written in **C++**.
It is created for learning, practicing problem-solving skills, and strengthening programming fundamentals.

The programs cover core concepts such as **sorting algorithms, stacks, queues, linked lists, and trees**. Each file demonstrates a specific algorithm or data structure with a simple implementation.

---

## Implemented Algorithms and Data Structures

### Sorting Algorithms

* Bubble Sort
* Insertion Sort
* Selection Sort
* Quick Sort
* Merge Sort

### Data Structures

* Linked List
* Stack
* Queue
* Binary Search Tree

### Other Implementations

* Merge operations
* Queue insertion operations
* Tree operations

## How to Compile and Run

1. Compile the program using a C++ compiler:

g++ filename.cpp -o program

2. Run the compiled program:

./program

Example:

g++ bubble_sort.cpp -o bubble
./bubble
